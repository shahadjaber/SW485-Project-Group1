Dataset Name: Online Shoppers Intention Dataset

Source (Kaggle URL):
https://www.kaggle.com/datasets/henrysue/online-shoppers-intention

License Information:
Kaggle Public Dataset (Check Kaggle page for specific license details).

Original Purpose:
This dataset was created to analyze online shopping behavior and predict whether a browsing session results in a purchase (Revenue = TRUE) or not (Revenue = FALSE). It contains information about user session attributes such as page visits, time spent, bounce rate, visitor type, and other related features.
